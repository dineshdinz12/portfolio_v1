# Modern Portfolio 1.0

A premium, high-performance professional portfolio built with Next.js 15, React, and GSAP. Featuring a minimalist design, smooth scroll interactions, and a fully responsive layout tailored for engineers and creatives.

## 🚀 Features

- **Next.js 15 (App Router)**: Leveraging the latest features for optimal performance and SEO.
- **GSAP & ScrollTrigger**: State-of-the-art scroll animations and interactive UI elements.
- **Tailwind CSS**: Modern styling with a deep amber/orange primary theme.
- **Responsive Design**: Mobile-first approach ensuring a stunning experience on all devices.
- **Dynamic Content**: Easy-to-manage project and achievement data via a centralized configuration.
- **Optimized Assets**: Optimized image loading and PDF handling for resumes.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [GSAP](https://greensock.com/gsap/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Fonts**: [Google Fonts (Anton, Roboto Flex, Playfair Display)](https://fonts.google.com/)

## 📦 Installation

To get a local copy up and running, follow these simple steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/dineshdinz12/portfolio_v1.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd portfolio_v1
   ```

3. **Install dependencies**
   ```bash
   npm install
   # or
   pnpm install
   ```

4. **Run the development server**
   ```bash
   npm run dev
   # or
   pnpm dev
   ```

5. **Build for production**
   ```bash
   npm run build
   ```
